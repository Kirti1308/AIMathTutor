# AIMathTutor
AI Math Tutor

## How to Run/Deploy on GitHub Pages

This project is ready to be deployed on GitHub Pages.

1. Go to the repository's **Settings** tab on GitHub.
2. In the left sidebar, click on **Pages** (under the "Code and automation" section).
3. Under **Build and deployment**, ensure **Deploy from a branch** is selected in the Source dropdown.
4. Under **Branch**, select `main` (or the current default branch) and ensure the folder is set to `/ (root)`.
5. Click **Save**.
6. Wait a few moments for the deployment to finish. You will see your live URL at the top of the page (e.g., `https://<your-username>.github.io/<repository-name>/`).
